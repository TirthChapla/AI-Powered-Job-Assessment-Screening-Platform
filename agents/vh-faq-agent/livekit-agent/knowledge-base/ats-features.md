# VoiceHire ATS Features (Currently Available)

## Job Management System

### Comprehensive Job Creation & Posting

- **Complete job information forms** with title, summary, responsibilities, and requirements
- **Job type configuration** (full-time, part-time, contract, internship, volunteer, temporary)
- **Experience level targeting** (entry, mid, senior, lead, executive)
- **Location settings** with country, city, and remote work options (remote, hybrid, flexible, office)
- **Industry and department categorization** using structured categories
- **Compensation fields** for salary information and benefits
- **Deadline management** with date/time picker for application deadlines
- **Job status control** (Draft, Published, On Hold, Internal, Closed)
- **Job duplication functionality** for creating similar positions quickly
- **Markdown editor support** for rich text content in job descriptions

### Advanced Job Management Dashboard

- **Unified job listing** with sortable columns (title, experience, location, compensation, status, date)
- **Advanced filtering** by job status with toggle for showing/hiding closed jobs
- **Real-time search functionality** across all job data fields
- **Pipeline stage visualization** with expandable row details showing candidate counts by stage
- **Color-coded status badges** for quick visual identification
- **Permission-based actions** (edit, share, preview) based on user roles
- **Responsive table design** with pagination and server-side data loading
- **Job sharing functionality** with dialog interface for team collaboration

### Interview Process Configuration

- **Custom interview stage creation** and management with categories
- **Drag-and-drop stage reordering** within predefined categories
- **Stage categorization** (Application, Screening, Interview, Decision, Offer, Onboarding)
- **Question set assignment** to specific interview stages
- **Evaluation form integration** for structured candidate assessment
- **Stage removal verification** to prevent accidental data loss
- **Color-coded stage visualization** for clear process mapping
- **Default stage protection** to maintain system integrity

## Candidate Management System

### Advanced Candidate Search & Discovery

- **Comprehensive candidate listing** with sortable columns for efficient browsing
- **Traditional full-text search** across candidate profiles and resumes
- **AI-powered semantic search** for context-based candidate discovery
- **Multi-criteria filtering** by status (Active, Archived, Do Not Contact)
- **Match scoring with detailed breakdowns** (similarity, work experience, education, skills)
- **Bookmark functionality** for important candidates
- **Application count tracking** per candidate across all jobs
- **Location display** with city, state, and country information
- **Server-side pagination** with configurable page sizes for performance

### Complete Candidate Profile Management

- **Create and edit candidate profiles** with comprehensive form validation
- **Email validation** with duplicate detection to maintain data integrity
- **Personal information management** (name, phone, email, location)
- **Professional information tracking** (current title, company, experience)
- **Tag system** for skills, technologies, and custom categorization
- **Resume upload and management** with version tracking
- **Status management** (Active, Archived, Do Not Contact)
- **Edit mode** with pre-populated data for efficient updates

### Talent Pool Organization

- **View candidate's current talent pool assignments** for strategic organization
- **Add candidates to multiple talent pools** simultaneously for pipeline management
- **Remove candidates from specific pools** with confirmation dialogs
- **Color-coded pool visualization** for easy identification
- **Pool assignment modal** with selection interface for bulk operations
- **Notes system** for pool assignments and candidate context
- **Pool membership history tracking** for relationship management

### Application History Tracking

- **Complete application tracking** across all jobs and time periods
- **Application status tracking** through all hiring stages
- **Job title and company information** display for context
- **Application date tracking** with chronological organization
- **Rating and match information** display for quick assessment
- **Resume information** for each application submitted
- **Stage category visualization** with color-coded badges
