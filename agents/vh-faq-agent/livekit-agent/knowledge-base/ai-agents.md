# VoiceHire AI Agents

## Overview

VoiceHire's revolutionary dual-agent approach combines AI-driven recruitment and interview capabilities to deliver a complete, automated hiring solution. Our AI agents work together to transform every aspect of the recruitment process from initial job posting to final candidate assessment.

## AI Agent Collaboration

### Seamless Workflow Integration

The Recruitment Agent and Interview Agent work together to create a complete automated hiring pipeline:

1. **Job Creation**: Recruitment Agent creates and distributes job postings
2. **Candidate Sourcing**: Identifies and engages qualified candidates
3. **Initial Screening**: Conducts AI phone screening for basic qualification
4. **Interview Handoff**: Qualified candidates are automatically scheduled with Interview Agent
5. **Comprehensive Assessment**: Interview Agent conducts in-depth evaluation
6. **Decision Support**: Both agents provide data and recommendations for final hiring decisions
