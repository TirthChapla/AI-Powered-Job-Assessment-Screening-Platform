# VoiceHire Pricing & Plans

## Simple, Transparent Pricing Philosophy

VoiceHire operates on a straightforward pay-as-you-use model with no hidden fees, long-term commitments, or complex contracts. Each service is priced separately, allowing organizations to choose exactly what they need and scale usage based on their hiring volume.

## Value Proposition by Service

### ATS Value Analysis

- **Cost per job post**: $10 (after pilot)
- **Typical savings per hire**: $500-2,000 through reduced time-to-fill and recruiter efficiency
- **ROI**: 50-200x return on investment for active hiring organizations
- **Break-even**: Typically achieved with first successful hire per job posting

### Interview Agent Value Analysis

- **Cost per hour**: $5
- **Typical interview duration**: 30-60 minutes ($2.50-$5.00 per interview)
- **Replacement cost**: $50-150 per traditional interview (recruiter time + coordination)
- **ROI**: 10-30x return through eliminated scheduling overhead and consistent evaluation
- **Additional benefits**: 24/7 availability, bias elimination, complete documentation

### Recruitment Agent Value Analysis

- **Cost per job**: $50
- **Typical agency fee**: 15-25% of annual salary ($15,000-50,000+ per hire)
- **ROI**: 300-1000x return compared to agency fees
- **Time savings**: 80% reduction in recruiter time per position
- **Quality improvement**: Consistent sourcing and screening processes
